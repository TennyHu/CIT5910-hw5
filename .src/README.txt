# SpellChecker and WordRecommender
## CIT 5910
## Author: Anya Cui, Tianyi Hu

### This homework exercise allows the user to input a text file and select a dictionary to use.
The program identifies any misspelled words and provides suggested corrections based on the chosen dictionary.
If no suggestions are available, the program prompts the user to either manually enter a replacement word or accept the
original spelling.

### Class
- SpellChecker.java
- SpellCheckerRunner.java
- Util.java
- WordRecommender.java

### How to use
- run SpellCheckerRunner.java
- follow the prompt, enter the dictinoary and the text file
- choose 'r' to select suggestions, 'a' to accept the misspelled word, 't' to manual replacement
